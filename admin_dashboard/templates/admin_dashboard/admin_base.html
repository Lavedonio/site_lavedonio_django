Finalmente chegou o grande momento de abrir o site para o mundo. Ainda tenho que testar diversas funcionalidades e garanto que há muita gambiarra potencialmente perigosa por trás, mas pelo menos você agora pode ver o que eu demorei tanto para fazer.

A ideia deste site é ser um repositório de projetos que eu faço e também ser uma ferramenta de testes de marketing digital, backend de desenvolvimento, entre outras.

Deveria estar estudando para o testinho de amanhã? Ou melhor, estar dormindo agora que estou exausto?
